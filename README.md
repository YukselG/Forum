# Forum

This is a full-stack forum web application built using React + TypeScript on the frontend and a RESTful API with C# and .NET on the backend, using SQL for the database. The app allows users to create accounts, make posts, and comment on existing posts, along with other community features. This project showcases full-stack development skills and includes features such as authentication and role-based authorization.

## Technologies used

-   Frontend: React, Typescript, React Router, CSS
-   Backend: C#, .NET, Entity Framework, ASP.NET Identity
-   Database: SQL Server (Relational Database Management System)
-   Other tools: Swagger, Visual Studio, Visual Studio Code, Git

## Features

### User Features:

-   User Registration & Authentication: Cookie-based authentication
-   Create Posts: Authenticated users can create posts within existing categories.
-   Comment on Posts: Users can comment on posts and engage in discussions.
-   View Posts & Comments: Users can view posts in each category and read/participate in comment threads.

### Admin Features:

-   Category Management: Only admins can create, edit, and delete categories.
-   User Management: Admins can manage users (e.g., deactivate users)
